# Flat UI Expert Skill

Use this skill whenever designing or implementing UI components in the EXFINOPS Flutter app that require a Flat Design aesthetic.

## Guidelines

1. **Minimalism & Clarity**: Remove unnecessary visual clutter (e.g., heavy drop shadows, gradients, bevels). Rely on 2D elements and sharp or slightly rounded corners.
2. **Vibrant & Solid Colors**: Use solid, distinctive, and vibrant color palettes consistent with the brand. Avoid complex gradients unless absolutely necessary for a specific subtle effect.
3. **Typography for Hierarchy**: Without shadows or depth, typography becomes the primary tool for visual hierarchy. Use distinct font sizes, weights, and colors to differentiate headings, body text, and actionable items.
4. **Iconography & Whitespace**: Use simple, easily recognizable flat icons. Employ generous whitespace (padding/margin) to group related elements logically without drawing boxes around them.
5. **Clear Call-to-Actions (CTAs)**: Buttons and interactive elements should stand out through contrasting solid colors rather than 3D effects. Hover/Tap states should be indicated by opacity changes or slight color shifts.
