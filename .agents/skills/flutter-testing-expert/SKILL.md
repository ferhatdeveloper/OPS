# Flutter Testing Expert Skill

Use this skill whenever writing tests for the EXFINOPS app.

## Guidelines

1. **Unit Testing Logic**: Write unit tests for ViewModels, Services, and Utilities. Focus on testing the outcomes of methods, edge cases, and error scenarios.
2. **Proper Mocking**: Use packages like `mockito` or `mocktail` to isolate the unit being tested. Do not make real network or database calls in unit tests.
3. **Widget Testing Framework**: When testing UI, ensure important user flows and specific UI states (Loading, Error, Success) are covered using `pumpWidget`.
4. **Given-When-Then Structure**: Structure tests clearly using the Given (Setup), When (Action), Then (Assertion) pattern to maintain readability.
